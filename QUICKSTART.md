# Quick Start Guide

## Build and Run xSnow in 30 Seconds

### 1. Build the app
```bash
cd /Users/kris.thomt/_/xsnow
make
```

### 2. Run it
```bash
make run
```

Or open it manually:
```bash
open build/xSnow.app
```

### 3. Control the snow
Look for the **snowflake icon** in your menu bar (top right). Click it to:
- Adjust snow intensity (Light/Medium/Heavy)
- Toggle wind effect on/off
- Toggle snow settling on windows on/off (enabled by default)
- Quit the app

## Install Permanently

To install xSnow to your Applications folder:
```bash
make install
```

Then launch it anytime from Spotlight or your Applications folder!

## What You'll See

- Beautiful falling snow particles on your desktop
- **Snow visibly piling up on top of your windows with realistic mounded shapes!**
- Snow can accumulate up to 80 pixels high before stopping
- Smooth, curved snow piles that look like real snow mounds
- Settled snow gradually melts away over 60 seconds
- Transparent overlay that doesn't block your work
- Mouse clicks pass through the snow
- Gentle wind effect for realistic movement
- Customizable intensity and optional settling

**Tip**: The app runs as a menu bar utility with no dock icon, keeping your workspace clean!
